# Word文档
## 安装 
`pip install python-docx`

## Word文档简介 
* Document：表示整个文档
* Paragraph：表示段落
* Run：表示相同样式文本

## 使用
`import docx
doc = docx.Document(filename)`

## Word样式
* 段落样式：可以应用于Paragraph
* 字符样式：可以应用于Run对象
* 链接样式：可以应用于Paragraph和Run对象
